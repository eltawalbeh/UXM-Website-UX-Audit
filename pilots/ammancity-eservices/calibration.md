# Pilot 3 calibration — Greater Amman Municipality E-Services Portal

- **Audit date:** 2026-07-15
- **Public scope:** Arabic catalog page only; desktop public browser at 1440 × 1200.
- **No-side-effect boundary:** No text was entered, no filters were changed, and no service, login, payment, or submission action was opened.
- **Evidence standard:** Each published finding has the original public source capture, a finding-specific annotated crop, capture timestamp, URL, page, and journey metadata.
- **Excluded verification:** Authenticated workflows, request forms, payment, validation, OTP, mobile, keyboard-only and automated accessibility testing were not evaluated.

## Score calibration

Six checkpoints were scored: 2 pass, 2 partial, and 2 issue. Using Pass = 1, Partial = 0.5, Issue = 0, the result is `(2 + 2×0.5) / 6 = 50%`. Three additional checkpoints are recorded as not verified and excluded from the denominator.

## Publication decision

Publish **2** findings only. Both observations are directly visible in the source screenshot and no finding relies on an interaction, inference about an untested workflow, or a missing capture.
